# Contributing to ARK

## Reporting Security Issues
This is a **Defensive Resilience tool.** If you find a "Jailbreak" that allows the AI to generate **Instructions for illegal or harmful activities (Violating the White Hat Constraint)**, **DO NOT OPEN A GITHUB ISSUE.**
* Email the Architect directly. - aneeshjoseph1091@gmail.com
* We treat these as Critical Patches.

## Suggesting Features
We accept Pull Requests for:
1.  New **Vulnerability Assessment Filters** (e.g., adding a specific cybersecurity framework).
2.  Optimizations for non-Gemini models (e.g., Claude compatibility).
